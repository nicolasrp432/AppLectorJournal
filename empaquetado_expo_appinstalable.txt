Desarrollar con Expo y React Native permite crear compilaciones nativas de forma muy sencilla. Para instalar la app en tu teléfono o subirla a las tiendas, necesitas salir del entorno de pruebas "Expo Go" y compilar tu código en un archivo ejecutable usando EAS (Expo Application Services).Opción 1: Instalar la app directamente en tu teléfonoPuedes instalar una versión nativa de tu app (archivo APK para Android o IPA para iOS) directamente en tu dispositivo sin necesidad de pasar por las tiendas de apps.Instala EAS CLI: En la terminal, ejecuta npm install -g eas-cli y luego inicia sesión con tu cuenta de Expo usando eas login.Configura tu proyecto: Ejecuta eas build:configure en la raíz de tu proyecto para inicializar la configuración.Crea la compilación de desarrollo/prueba:Para Android: Ejecuta eas build --profile development --platform android.Para iOS: Ejecuta eas build --profile development --platform ios.Instala en el dispositivo:Cuando el proceso termine (tarda unos minutos), se generará un enlace.Ábrelo desde el navegador de tu teléfono móvil y descarga la aplicación directamente.Para una guía detallada y paso a paso sobre cómo realizar tus compilaciones (builds) con EAS:15mReact Native Development Build tutorial with EASDave GrayYouTube• 22 nov 2024Opción 2: Subirla a Apple App Store y Google Play StorePara publicar tu aplicación oficialmente, debes compilar un paquete de producción y enviarlo a las tiendas.Requisitos previos: Necesitas una cuenta de desarrollador en Google Play Console (pago único de \\$25) y en Apple Developer Program (\\$99/año).Generar la compilación: Ejecuta eas build --platform all (o especifica iOS o Android). Expo compilará tu app en la nube y gestionará todas las firmas criptográficas necesarias de forma automática.Enviar a las tiendas:La forma más rápida de subir el archivo generado es usando EAS Submit.Simplemente ejecuta eas submit --platform android y eas submit --platform ios, e introduce los datos de tus cuentas de desarrollador.Alternativamente, puedes descargar el archivo compilado (formato AAB para Android o IPA para iOS) desde el panel de control de Expo y subirlo manualmente a las consolas de Google y Apple.
